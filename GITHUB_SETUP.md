# GitHub Setup Instructions

## Quick Setup

### Option 1: If you already have a GitHub repository

Run this command (replace YOUR_USERNAME with your GitHub username):

```bash
git remote add origin https://github.com/YOUR_USERNAME/rideokinawa.git
git branch -M main
git push -u origin main
```

### Option 2: Create a new GitHub repository

1. **Create Repository on GitHub:**
   - Go to: https://github.com/new
   - Repository name: `rideokinawa`
   - Choose **Public** or **Private**
   - **DO NOT** check any boxes (no README, .gitignore, or license)
   - Click **"Create repository"**

2. **After creating, GitHub will show you commands. Use these:**
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/rideokinawa.git
   git branch -M main
   git push -u origin main
   ```

Replace `YOUR_USERNAME` with your actual GitHub username.
